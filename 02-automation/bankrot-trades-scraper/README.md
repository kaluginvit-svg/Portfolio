# Парсер торгов bankrot.fedresurs.ru + обзор SQLite

**Что это:** сбор объявлений о торгах с публикаций Федресурса, сохранение в **SQLite**, вспомогательные скрипты фильтрации и просмотра; есть веб-просмотр (`db_review_server.py`) для работы с базой.

**Стек:** Python, SQLite, парсинг через `scrapling` / HTTP.

**Запуск:** зависимости из окружения проекта (см. импорты в `fetch_bankrot_trades.py`); типичный сценарий — запуск парсера с аргументами командной строки и последующий просмотр через маленький Flask/UI-сервер при наличии.

**Статус:** автоматизация сбора **публичных** данных; соблюдайте robots/правила источника и нагрузку на сайт.
