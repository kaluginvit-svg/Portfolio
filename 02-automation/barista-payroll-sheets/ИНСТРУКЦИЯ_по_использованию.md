# Инструкция по использованию скрипта автоматизации зарплаты бариста

## Подготовка

1. Откройте вашу таблицу Google Sheets "Зарплата_бариста_Декабрь_2025"
2. Перейдите в меню: **Расширения → Apps Script**
3. Удалите весь код по умолчанию
4. Скопируйте содержимое файла `Автоматизация_зарплаты_бариста.gs` и вставьте в редактор Apps Script
5. Сохраните проект (Ctrl+S или Cmd+S)

## Первоначальная настройка

### Шаг 1: Проверьте структуру таблицы

Убедитесь, что ваша таблица имеет следующие листы:
- **"Смены"** - с колонками: Дата, Сотрудник, Время начала, Время завершения, Часы работы
- **"Зарплата_период"** - с колонками: Сотрудник, Часы работы, Выручка, Зарплата
- Листы по месяцам (например, "декабрь", "январь") - с данными о выручке

**Важно:** Названия колонок должны точно совпадать с указанными выше (регистр не важен).

### Шаг 2: Настройка констант

Если названия ваших листов или колонок отличаются, отредактируйте константы в начале скрипта:

```javascript
const SHEET_NAMES = {
  SHIFTS: 'Смены',              // Название листа со сменами
  SALARY_PERIOD: 'Зарплата_период'  // Название листа с зарплатой
};

const COLUMN_NAMES = {
  DATE: 'Дата',
  EMPLOYEE: 'Сотрудник',
  START_TIME: 'Время начала',
  END_TIME: 'Время завершения',
  HOURS: 'Часы работы',
  REVENUE: 'Выручка',
  SALARY: 'Зарплата'
};
```

### Шаг 3: Первый запуск

1. Вернитесь в таблицу Google Sheets
2. Обновите страницу (F5)
3. В меню появится новый пункт: **"Автоматизация зарплаты"**
4. Выберите: **Автоматизация зарплаты → Настроить таблицу**

Скрипт автоматически:
- Настроит выпадающие списки на листе "Смены"
- Установит все необходимые формулы
- Создаст связи между листами

## Использование

### Меню "Автоматизация зарплаты"

После настройки в меню доступны следующие функции:

1. **Настроить таблицу** - полная инициализация (используйте при первом запуске)
2. **Обновить выпадающие списки** - обновляет списки сотрудников и времени
3. **Обновить формулы** - переустанавливает все формулы
4. **Добавить нового сотрудника** - добавляет сотрудника в "Зарплата_период" с автоматическими формулами
5. **Пересчитать зарплату** - пересчитывает все данные

### Работа с выпадающими списками

На листе **"Смены"**:
- Колонка **"Сотрудник"** - выпадающий список всех сотрудников из таблицы
- Колонка **"Время начала"** - выпадающий список времени с шагом 30 минут (00:00, 00:30, 01:00...)
- Колонка **"Время завершения"** - аналогично времени начала

**Автоматическое обновление:** При добавлении нового сотрудника в любом месте таблицы, список автоматически обновится.

### Добавление нового сотрудника

**Способ 1 (через меню):**
1. Выберите: **Автоматизация зарплаты → Добавить нового сотрудника**
2. Введите имя сотрудника
3. Скрипт автоматически добавит строку с формулами

**Способ 2 (вручную):**
1. Добавьте новую строку в лист "Зарплата_период"
2. Введите имя сотрудника в колонку "Сотрудник"
3. Формулы применятся автоматически (благодаря функции `onEdit`)

### Добавление нового месяца

1. Создайте новый лист с названием месяца (например, "февраль")
2. Убедитесь, что структура колонок совпадает с другими месяцами
3. Выберите: **Автоматизация зарплаты → Обновить формулы**
4. Скрипт автоматически подключит новый лист к расчетам

## Как работают формулы

### Принцип работы

Скрипт использует **поиск по заголовкам**, а не по номерам колонок. Это означает:

✅ **Работает:**
- Изменение порядка колонок
- Добавление новых колонок
- Удаление ненужных колонок (кроме используемых)

❌ **Не работает:**
- Изменение названий колонок (без обновления констант)
- Удаление используемых колонок

### Примеры формул

**Расчет часов работы (лист "Смены"):**
```
=IF(AND(Время начала<>"", Время завершения<>""), 
  (TIMEVALUE(Время завершения) - TIMEVALUE(Время начала)) * 24, "")
```

**Суммирование часов по сотруднику (лист "Зарплата_период"):**
```
=SUMIF('Смены'!Сотрудник:Сотрудник, A2, 'Смены'!Часы работы:Часы работы)
```

**Суммирование выручки из всех месяцев:**
```
=SUMIF('декабрь'!Сотрудник:Сотрудник, A2, 'декабрь'!Выручка:Выручка) + 
 SUMIF('январь'!Сотрудник:Сотрудник, A2, 'январь'!Выручка:Выручка)
```

## Логирование

Скрипт автоматически создает лист **"Логи"** для отслеживания всех действий:
- Что было сделано
- Сколько строк обработано
- Где возникли ошибки

Логи хранятся последние 1000 записей.

## Решение проблем

### Проблема: "Лист не найден"
**Решение:** Проверьте названия листов. Они должны точно совпадать с константами `SHEET_NAMES`.

### Проблема: "Колонка не найдена"
**Решение:** 
1. Проверьте названия колонок в первой строке
2. Убедитесь, что они совпадают с константами `COLUMN_NAMES`
3. Проверьте, нет ли лишних пробелов в названиях

### Проблема: Формулы не работают
**Решение:**
1. Выберите: **Автоматизация зарплаты → Обновить формулы**
2. Проверьте лист "Логи" на наличие ошибок
3. Убедитесь, что структура всех листов одинаковая

### Проблема: Выпадающие списки не обновляются
**Решение:**
1. Выберите: **Автоматизация зарплаты → Обновить выпадающие списки**
2. Убедитесь, что в таблице есть хотя бы один сотрудник

### Проблема: Меню не появляется
**Решение:**
1. Обновите страницу (F5)
2. Убедитесь, что скрипт сохранен
3. Проверьте, что функция `onOpen()` присутствует в скрипте

## Настройка под свои нужды

### Изменение формулы расчета зарплаты

Найдите функцию `setupSalaryPeriodFormulas()` и измените строку:

```javascript
const formula = `=${hoursCell} * 500`; // Замените 500 на вашу ставку
```

Например, если ставка хранится в колонке "Ставка":
```javascript
const rateCol = headers['Ставка'];
const rateCell = getColumnLetter(rateCol) + i;
const formula = `=${hoursCell} * ${rateCell}`;
```

### Изменение шага времени

В функции `generateTimeOptions()` измените шаг:

```javascript
for (let minute = 0; minute < 60; minute += 30) { // 30 минут
```

Например, для шага 15 минут:
```javascript
for (let minute = 0; minute < 60; minute += 15) { // 15 минут
```

### Добавление новых колонок

1. Добавьте название колонки в константы `COLUMN_NAMES`
2. Добавьте логику обработки в соответствующие функции
3. Обновите формулы через меню

## Безопасность

- Скрипт работает только с вашей таблицей
- Все изменения логируются
- Можно откатить изменения через историю Google Sheets
- Рекомендуется сделать копию таблицы перед первым запуском

## Поддержка

При возникновении проблем:
1. Проверьте лист "Логи"
2. Откройте консоль разработчика (F12) для просмотра ошибок
3. Убедитесь, что структура таблицы соответствует требованиям
