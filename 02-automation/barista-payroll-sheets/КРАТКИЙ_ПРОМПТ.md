# Краткий промпт для создания Apps Script

Создай Apps Script для Google Sheets для автоматизации таблицы расчета зарплаты бариста со следующими требованиями:

## Основные принципы:
1. **Не завязан на номера колонок** - используй поиск по заголовкам через функцию `MATCH()` и `findHeaders()`
2. **Смысловые якоря** - работай с названиями листов, заголовками, параметрами (не с A1, B2 и т.д.)
3. **Пакетная обработка** - обрабатывай диапазоны данных целиком через `setFormulas()`, `setDataValidation()` для диапазонов
4. **Понятные сообщения** - логируй что сделал, сколько строк обработал, где проблема в функцию `logActivity()` и лист "Логи"

## Структура таблицы:

### Лист "Смены":
- **Дата** - дата смены
- **Сотрудник** - имя сотрудника (выпадающий список)
- **Время начала** - время начала смены (выпадающий список, шаг 30 мин)
- **Время завершения** - время завершения смены (выпадающий список, шаг 30 мин)
- **Часы работы** - автоматический расчет: `(Время завершения - Время начала) * 24`

### Листы месяцев ("декабрь", "январь" и т.д.):
- **Дата** - дата
- **Сотрудник** - имя сотрудника
- **Выручка** - сумма выручки за день

### Лист "Зарплата_период":
- **Сотрудник** - имя сотрудника
- **Часы работы** - сумма часов из листа "Смены" по сотруднику: `SUMIF(Смены!Сотрудник, A2, Смены!Часы работы)`
- **Выручка** - сумма выручки из всех листов месяцев: `SUMIF(декабрь!Сотрудник, A2, декабрь!Выручка) + SUMIF(январь!Сотрудник, A2, январь!Выручка) + ...`
- **Зарплата** - расчет: `Часы работы * Ставка` (если есть колонка "Ставка") или фиксированная ставка

## Функциональность:

### 1. Выпадающие списки на листе "Смены":
- **Колонка "Сотрудник"**: 
  - Собирать всех уникальных сотрудников из листов "Смены" и "Зарплата_период"
  - Создать `DataValidation` с `requireValueInList()`
  - Применить к диапазону строк (минимум 100 строк для будущих данных)
  - Автоматически обновлять при добавлении новых сотрудников
  
- **Колонки "Время начала" и "Время завершения"**:
  - Генерировать список времени с шагом 30 минут: 00:00, 00:30, 01:00, ..., 23:30
  - Создать `DataValidation` с этим списком
  - Применить к диапазону строк

### 2. Автоматизация формул:
- **Поиск колонок**: Функция `findHeaders(sheet)` должна:
  - Читать первую строку листа
  - Создавать объект `{ "Название колонки": номер_колонки }`
  - Поддерживать поиск в нижнем регистре для гибкости
  
- **Формулы на листе "Смены"**:
  - Колонка "Часы работы": `=IF(AND(Время начала<>"", Время завершения<>""), (TIMEVALUE(Время завершения) - TIMEVALUE(Время начала)) * 24, "")`
  - Применять формулу ко всем строкам с данными пакетно через `setFormulas()`
  
- **Формулы на листе "Зарплата_период"**:
  - Колонка "Часы работы": `=SUMIF('Смены'!Сотрудник:Сотрудник, A2, 'Смены'!Часы работы:Часы работы)`
  - Колонка "Выручка": Динамически собирать формулу из всех листов месяцев
  - Колонка "Зарплата": Использовать колонку "Ставка" если есть, иначе фиксированное значение

### 3. Масштабируемость:
- **Добавление нового месяца**:
  - Функция `getMonthSheets()` должна находить все листы кроме "Смены" и "Зарплата_период"
  - При обновлении формул автоматически включать новые листы в расчет выручки
  
- **Добавление нового сотрудника**:
  - Функция `addNewEmployee()` добавляет строку в "Зарплата_период"
  - Копирует формулы из предыдущей строки
  - Обновляет выпадающие списки
  
- **Автоматическое применение формул**:
  - Функция `onEdit(e)` отслеживает добавление новых строк
  - При добавлении сотрудника в новую строку автоматически копирует формулы

### 4. Обработка ошибок и валидация:
- Функция `checkTableStructure()` проверяет:
  - Наличие необходимых листов
  - Наличие необходимых колонок
  - Выводит понятные сообщения об ошибках
  
- Все функции должны быть обернуты в `try-catch`
- Логировать ошибки с указанием функции и места проблемы

## Функции скрипта:

### Основные функции:
- `onOpen()` - создание меню "Автоматизация зарплаты" с пунктами:
  - Настроить таблицу
  - Обновить выпадающие списки
  - Обновить формулы
  - Добавить нового сотрудника
  - Пересчитать зарплату

- `initializeTable()` - полная инициализация:
  - Проверка структуры через `checkTableStructure()`
  - Настройка выпадающих списков
  - Установка формул
  - Обновление списка сотрудников

- `setupDataValidation()` - настройка выпадающих списков:
  - Получение списка сотрудников через `getEmployeeList()`
  - Генерация списка времени через `generateTimeOptions()`
  - Применение валидации к диапазонам

- `setupFormulas()` - установка формул:
  - `setupShiftsFormulas()` - формулы на листе "Смены"
  - `setupMonthSheetsFormulas()` - формулы на листах месяцев
  - `setupSalaryPeriodFormulas()` - формулы на листе "Зарплата_период"

### Вспомогательные функции:
- `findHeaders(sheet)` - поиск заголовков и создание объекта с номерами колонок
- `getColumnLetter(columnNumber)` - преобразование номера колонки в букву (A, B, ..., Z, AA, AB...)
- `getEmployeeList()` - сбор всех уникальных сотрудников из таблицы
- `getMonthSheets()` - получение списка листов месяцев
- `generateTimeOptions()` - генерация списка времени с шагом 30 минут
- `checkTableStructure()` - проверка структуры таблицы
- `ensureHeaders(sheet)` - создание заголовков если их нет
- `addNewEmployee()` - добавление нового сотрудника через диалог
- `calculateSalary()` - пересчет всех данных
- `onEdit(e)` - автоматическое применение формул при редактировании
- `logActivity(message)` - логирование в лист "Логи" с временной меткой

## Примеры формул (с поиском по заголовкам):

```javascript
// ❌ ПЛОХО: Привязано к конкретной колонке
=SUM(B2:B10)
=VLOOKUP(A2, декабрь!A:B, 2, FALSE)

// ✅ ХОРОШО: Поиск по заголовкам
// Сумма часов работы (находит колонку "Часы работы" автоматически)
=SUMIF('Смены'!Сотрудник:Сотрудник, A2, 'Смены'!Часы работы:Часы работы)

// Сумма выручки из всех месяцев (динамически находит листы)
=SUMIF('декабрь'!Сотрудник:Сотрудник, A2, 'декабрь'!Выручка:Выручка) + 
 SUMIF('январь'!Сотрудник:Сотрудник, A2, 'январь'!Выручка:Выручка)

// Расчет часов работы (использует названия колонок)
=IF(AND(Время начала<>"", Время завершения<>""), 
    (TIMEVALUE(Время завершения) - TIMEVALUE(Время начала)) * 24, "")
```
