- [ ] GCP: проект, включён **Google Sheets API**; скачан **JSON сервисного аккаунта**.
- [ ] Таблица: доступ **Редактор** для `client_email` из JSON сервисного аккаунта.
- [ ] В проекте есть [`google_sheets.py`](google_sheets.py) с операциями через API (чтение, запись, очистка, при необходимости append).
- [ ] Есть [`report_generator.py`](report_generator.py) на **tkinter**, генерирует случайные данные и пишет через модуль.
- [ ] Реализовано **форматирование** (цвет/шрифт/объединение) через `batchUpdate` в `GoogleSheets.format_report_sheet`.
