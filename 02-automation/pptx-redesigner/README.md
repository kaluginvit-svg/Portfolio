# Утилиты для разбора и редизайна PPTX

**Что это:** набор скриптов в [`tools/`](./tools/) для анализа структуры презентаций (**извлечение макетов, цветов, фигур, картинок**) и генерации **нового PPTX** по правилам редизайна (`redesign_pptx.py`, `redesign_pptx_v2.py`).

**Стек:** Python, `python-pptx` и связанные зависимости (см. импорты в скриптах).

**Запуск:** из каталога `tools/` или с указанием пути к входному `.pptx`; детали — в docstring соответствующего файла.

**Статус:** автоматизация подготовки слайдов под корпоративный шаблон; проверяйте результат вручную перед важными презентациями.
