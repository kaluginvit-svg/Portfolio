В **Документации** есть отдельная вкладка **Установка**.

**Haystack** позволяет создать рабочего агента всего несколькими строками — аналогично тому, как это делалось в **LangChain**.

В примере показывается код: импортируется модуль `OS`, затем несколько классов из `haystack.agent`, включая генератор `OpenAI`, сообщения чата `ChatMessage`, компонент `Tool`.

```
import os
from haystack.components.agents import Agent
from haystack. components.generators.chat import OpenAIChatGenerator
from haystack.dataclasses import ChatMessage
from haystack. tools import ComponentTool
from haystack.components.websearch import SerperDevWebSearch
os. environ["OPENAI_API_KEY"] = "<YOUR OPENAI API KEY>"
os. environ["SERPERDEV_API_KEY"] = "<YOUR SERPERDEV API KEY>"
search_tool = ComponentTool(component=SerperDevWebSearch())
basic_agent = Agent(
chat_generator=OpenAIChatGenerator(model="gpt-4o-mini"),
system_prompt="You are a helpful web agent.",
tools=[search_tool],
)
result = basic_agent.run(messages=[ChatMessage. from_user("When was the first version of Haystack released?"
print(result['last_message' ]. text)
```

Отдельный компонент — `SerperDevWebSearch`. Это стороннее приложение, позволяющее выполнять гугл-поиск.
