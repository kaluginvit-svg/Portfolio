# Finance MCP Server

**Что это:** MCP-сервер и опциональный UI для финансовых сценариев в ИИ-ассистентах (учёт, справки по продуктам и т.д. — см. код подпроектов).

**Структура:**

| Папка | Описание |
|--------|-----------|
| [`product-mcp/`](./product-mcp/) | MCP-сервер (основная документация по установке) |
| [`product-mcp-ui/`](./product-mcp-ui/) | Простой UI для проверки |

**Запуск:** следуйте README в **`product-mcp/`**; для Cursor добавьте сервер в конфиг MCP по аналогии с другими MCP в портфолио.

**Статус:** интеграционный кейс; секреты только в `.env`, не в git.
