# Практические примеры использования PyTrends MCP

## Примеры для SEO-задач

### 1. Анализ популярности ключевых слов

**Задача:** Сравнить популярность нескольких ключевых слов за год

```python
# В Cursor чате просто напишите:
"Сравни популярность 'seo оптимизация', 'продвижение сайта', 'контекстная реклама' за последние 12 месяцев в России"

# Или вызовите напрямую:
interest_over_time(
    keywords="seo оптимизация, продвижение сайта, контекстная реклама",
    timeframe="today 12-m",
    geo="RU",
    hl="ru-RU"
)
```

---

### 2. Географический анализ интереса

**Задача:** Узнать, в каких странах больше всего интересуются вашей темой

```python
# В Cursor чате:
"Покажи популярность 'artificial intelligence' по странам за последние 5 лет"

# Прямой вызов:
interest_by_region(
    keywords="artificial intelligence",
    resolution="COUNTRY",
    timeframe="today 5-y",
    geo=""
)
```

---

### 3. Поиск связанных тем для контента

**Задача:** Найти темы для статей, связанные с вашей ключевой темой

```python
# В Cursor чате:
"Найди связанные темы для 'machine learning'"

# Прямой вызов:
related_topics(
    keywords="machine learning",
    timeframe="today 5-y",
    geo="US"
)
```

**Результат покажет:**
- Rising topics (растущие темы)
- Top topics (популярные темы)

---

## Примеры запросов в Cursor (естественный язык)

Вы можете просто писать запросы на русском языке, и AI автоматически вызовет нужные инструменты:

1. **"Покажи популярность Python и JavaScript за последний год"**
   → `interest_over_time(keywords="python, javascript", timeframe="today 12-m")`

2. **"В каких странах больше всего интересуются машинным обучением?"**
   → `interest_by_region(keywords="machine learning", resolution="COUNTRY")`

3. **"Найди связанные темы для SEO"**
   → `related_topics(keywords="seo")`

4. **"Что сейчас в тренде в России?"**
   → `trending_searches(pn="russia", hl="ru-RU")`

5. **"Дай предложения для ключевого слова 'веб-разработка'"**
   → `suggestions(keyword="веб-разработка", hl="ru-RU")`

## Полезные параметры

### Timeframe (период времени):
- `'today 5-y'` - последние 5 лет (максимум)
- `'today 12-m'` - последние 12 месяцев
- `'today 3-m'` - последние 3 месяца
- `'today 1-m'` - последний месяц
- `'now 7-d'` - последние 7 дней
- `'now 1-d'` - последние 24 часа

### Geo (коды стран):
- `''` - весь мир
- `'US'` - США
- `'RU'` - Россия
- `'GB'` - Великобритания
- `'DE'` - Германия
- `'FR'` - Франция
- И другие коды ISO 3166-1 alpha-2

### Resolution (для interest_by_region):
- `'COUNTRY'` - по странам
- `'REGION'` - по регионам
- `'CITY'` - по городам
- `'DMA'` - по DMA
- `'METRO'` - по метрополиям

## Советы по использованию

1. **Делайте паузы** между запросами (1-2 секунды), чтобы избежать блокировок
2. **Используйте естественный язык** в Cursor - AI сам выберет нужный инструмент
3. **Комбинируйте инструменты** для комплексного анализа
4. **Сохраняйте результаты** для дальнейшего анализа
