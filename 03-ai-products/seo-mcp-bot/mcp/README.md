# PyTrends MCP Server

MCP (Model Context Protocol) сервер для работы с Google Trends через библиотеку PyTrends.

## Расположение

Сервер находится в папке `mcp/` проекта SEO-бот.

## Установка

```bash
cd mcp
pip install -r requirements.txt
```

## Настройка в Cursor

Сервер настроен в `mcp.json`:
- Путь: `C:\Users\Виталий\.cursor\mcp.json`

## Возможности

1. **interest_over_time** - Получить данные о популярности ключевых слов за период времени
2. **interest_by_region** - Получить данные о популярности по регионам
3. **related_topics** - Найти связанные темы для ключевых слов
4. **related_queries** - Найти связанные поисковые запросы
5. **trending_searches** - Получить текущие трендовые поисковые запросы
6. **top_charts** - Получить топ чарты Google Trends
7. **suggestions** - Получить предложения ключевых слов (автодополнение)

## Использование

После перезапуска Cursor, инструменты PyTrends будут доступны через MCP протокол.

## Быстрый старт

### Пример 1: Сравнить популярность ключевых слов

```

Найди связанные темы для 'machine learning'
```
