# Промпт для тестирования PyTrends MCP

## После перезапуска Cursor используйте этот промпт:

```
Протестируй PyTrends MCP сервер. Выполни следующие запросы по порядку:

1. Используй инструмент suggestions для ключевого слова "python" с языком en-US
2. Подожди 3 секунды
3. Используй interest_over_time для ключевых слов "python, javascript" за последние 3 месяца (today 3-m) в США (geo=US, hl=en-US)
4. Подожди 3 секунды  
5. Используй related_topics для ключевого слова "python" за последние 12 месяцев в США
6. Подожди 3 секунды
7. Используй trending_searches для региона "united_states" с языком en-US

Покажи результаты каждого запроса и сообщи, если возникли ошибки.
```

---

## Альтернативный упрощенный промпт:

```
Протестируй PyTrends MCP: 
1. Получи предложения для "python" 
2. Подожди 3 секунды
3. Получи популярность "python" за последние 3 месяца в США
4. Покажи результаты
```

---

## Промпт для проверки конкретного инструмента:

```
Используй PyTrends MCP инструмент interest_over_time с параметрами:
- keywords: "python, javascript"
- timeframe: "today 3-m"
- geo: "US"
- hl: "en-US"

Покажи результат.
```
