# Команды PyTrends MCP Server

## Запуск сервера

Сервер запускается автоматически Cursor через команду из `mcp.json`:

```bash
python mcp/server.py
```

## Доступные инструменты (Tools)

После перезапуска Cursor, вы можете вызывать эти инструменты через MCP протокол. Вот примеры использования:

### 1. interest_over_time

Получить данные о популярности ключевых слов за период времени.

**Пример вызова:**
```
interest_over_time(
    keywords="python, javascript",
    timeframe="today 12-m",
    geo="US",
    hl="en-US"
)
```

**Параметры:**
- `keywords` (обязательный) - Ключевые слова через запятую: "python, javascript, java"
- `timeframe` (опционально) - Период: 'today 5-y', 'today 12-m', 'today 3-m', 'today 1-m', 'now 7-d', 'now 1-d'
- `geo` (опционально) - Код страны: 'US', 'RU', 'GB', '' (весь мир)
- `hl` (опционально) - Язык: 'en-US', 'ru-RU'
- `tz` (опционально) - Часовой пояс в минутах от UTC (по умолчанию 360)

---

### 2. interest_by_region

Получить данные о популярности по регионам.

**Пример вызова:**
```
interest_by_region(
    keywords="artificial intelligence",
    resolution="COUNTRY",
    timeframe="today 5-y",
    geo=""
)
```

**Параметры:**
- `keywords` (обязательный) - Ключевые слова через запятую
- `resolution` (опционально) - 'COUNTRY', 'REGION', 'CITY', 'DMA', 'METRO'
- `timeframe` (опционально) - Период времени
- `geo` (опционально) - Географический регион
- `hl` (опционально) - Язык интерфейса
- `tz` (опционально) - Часовой пояс

---

### 3. related_topics

Найти связанные темы для ключевого слова.

**Пример вызова:**
```
related_topics(
    keywords="machine learning",
    timeframe="today 5-y",
    geo="US"
)
```

**Параметры:**
- `keywords` (обязательный) - Одно ключевое слово
- `timeframe` (опционально) - Период времени
- `geo` (опционально) - Географический регион
- `hl` (опционально) - Язык интерфейса
- `tz` (опционально) - Часовой пояс

---

### 4. related_queries

Найти связанные поисковые запросы.

**Пример вызова:**
```
related_queries(
    keywords="python programming",
    timeframe="today 5-y",
    geo=""
)
```

**Параметры:**
- `keywords` (обязательный) - Одно ключевое слово
- `timeframe` (опционально) - Период времени
- `geo` (опционально) - Географический регион
- `hl` (опционально) - Язык интерфейса
- `tz` (опционально) - Часовой пояс

---

### 5. trending_searches

Получить текущие трендовые поисковые запросы.

**Пример вызова:**
```
trending_searches(
    pn="united_states",
    hl="en-US"
)
```

**Параметры:**
- `pn` (опционально) - Название страны: 'united_states', 'russia', 'germany'
- `hl` (опционально) - Язык интерфейса

---

### 6. top_charts

Получить топ чарты Google Trends.

**Пример вызова:**
```
top_charts(
    date="202401",
    geo="US",
    hl="en-US"
)
```

**Параметры:**
- `date` (опционально) - Дата в формате YYYYMM: '202401' (январь 2024). Пустая строка = текущий год
- `geo` (опционально) - Код страны: 'US', 'RU'
- `hl` (опционально) - Язык интерфейса
- `tz` (опционально) - Часовой пояс

---

### 7. suggestions

Получить предложения ключевых слов (автодополнение).

**Пример вызова:**
```
suggestions(
    keyword="python programming",
    hl="en-US"
)
```

**Параметры:**
- `keyword` (обязательный) - Ключевое слово для поиска предложений
- `hl` (опционально) - Язык интерфейса

---

## Использование в Cursor

После перезапуска Cursor, вы можете использовать эти инструменты:

1. **Через чат с AI** - просто опишите, что вам нужно, и AI вызовет соответствующий инструмент
2. **Прямой вызов** - в чате можно явно указать название инструмента и параметры

**Пример запроса в Cursor:**
```
Покажи популярность "python" и "javascript" за последние 12 месяцев в США
```

AI автоматически вызовет:
```
interest_over_time(keywords="python, javascript", timeframe="today 12-m", geo="US")
```

---

## Формат ответа

Все инструменты возвращают JSON строку с результатами:

```json
{
  "keywords": ["python", "javascript"],
  "timeframe": "today 12-m",
  "geo": "US",
  "data": {
    "2024-01-01": {"python": 85, "javascript": 100},
    "2024-02-01": {"python": 90, "javascript": 95}
  }
}
```

---

## Примечания

⚠️ **Ограничения:**
- Google может ограничивать количество запросов
- Делайте паузы между запросами (не менее 1 секунды)
- Не делайте слишком много запросов подряд
