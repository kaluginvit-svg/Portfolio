# Как получить OAuth токен для Yandex Wordstat API

## Быстрая инструкция (5 минут)

### 1. Откройте страницу создания OAuth приложения
👉 **https://oauth.yandex.ru/**

### 2. Нажмите "Создать новое приложение"

### 3. Заполните форму:

**Общие данные:**
- Название: `Wordstat MCP Server`
- Описание: `MCP сервер для SEO анализа`
- Платформы: выберите **"Веб-сервисы"** или **"Другое"**

**Доступ к данным:**
- Найдите и отметьте: **"Использование API Вордстата"** или **"Wordstat API"**
  (Может быть в разделе "Доступ к API Яндекс Директа")

### 4. Нажмите "Создать приложение"

### 5. Получите токен:
- После создания найдите кнопку **"Создать новый токен"** или **"Получить токен"**
- Выберите права: **"Wordstat API"**
- Нажмите **"Создать"**
- **Скопируйте токен** (начинается с `y0_Ag...`) - он показывается только один раз!

### 6. Вставьте токен в файл `.env`

Откройте файл `.env` в папке проекта:
```bash
notepad yandex-wordstat-mcp\.env
```

Замените строку:
```
WORDSTAT_TOKEN=ВАШ_OAUTH_ТОКЕН_ОТ_ЯНДЕКСА
```

На:
```
WORDSTAT_TOKEN=y0_AgAABBBCCC111222333...
```
(вставьте ваш реальный токен)

### 7. Подайте заявку на доступ к API

⚠️ **Важно:** После создания приложения нужно подать заявку:

1. Откройте: **https://yandex.ru/support/direct/**
2. Выберите тему: **"API"**
3. В сообщении укажите:
   ```
   Прошу предоставить доступ к API Wordstat.
   
   Логин: [ваш логин Яндекс]
   Client ID: [найдите на странице приложения]
   ```
4. Отправьте и дождитесь ответа (1-3 дня)

### 8. Проверьте настройку

```bash
cd yandex-wordstat-mcp
python check_setup.py
```
