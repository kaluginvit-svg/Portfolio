# Telegram-бот погоды

Бот для получения текущей погоды, прогноза на 5 дней, сравнения городов, расширенных данных (погода + качество воздуха), уведомлений по расписанию. Поддерживает кэш ответов OpenWeather (10 мин) и inline-режим (поиск погоды по городу из любого чата).

## Технологии

- Python 3
- **requests** — запросы к API
- **python-dotenv** — переменные окружения из `.env`
- **pyTelegramBotAPI** — Telegram Bot API
- **OpenWeather API** — погода, геокодинг, прогноз, качество воздуха

## Установка

1. Клонируйте или скопируйте репозиторий.
2. Создайте виртуальное окружение и установите зависимости:

   ```bash
   python -m venv venv
   venv\Scripts\activate   # Windows
   # source venv/bin/activate   # Linux/macOS
   pip install -r requirements.txt
   ```

3. Настройте `.env` на основе `.env.example`:
   - **OW_API_KEY** — ключ [OpenWeather API](https://openweathermap.org/api)
   - **BOT_TOKEN** — токен бота от [@BotFather](https://t.me/BotFather)

## Запуск

```bash
python bot.py
```

## Основные команды и меню

| Действие | Описание |
|----------|----------|
| **/start** | Приветствие и главное меню с кнопками |
| **Текущая погода** | Погода по сохранённому городу/геолокации или ввод города |
| **Прогноз на 5 дней** | Выбор дня → слоты по 3 часа (нужны координаты или геолокация) |
| **Моя геолокация** | Отправка координат и сохранение для последующих запросов |
| **Сравнить города** | Ввод двух городов через запятую → таблица температура/описание |
| **Расширенные данные** | Погода + качество воздуха по компонентам (нужны координаты) |
| **Уведомления** | Включение/выключение и выбор интервала (1, 2, 3, 6 ч) |

**Inline-режим:** в любом чате введите `@ваш_бот город` — появится результат с погодой и ссылкой на прогноз (в BotFather включите Inline Mode: `/setinline`).
