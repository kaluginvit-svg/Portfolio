# Тестовый модуль для работы с API

Проект для тестирования различных API с консольным и веб-интерфейсом.

## Возможности

- **GET запрос по URL** - выполнение произвольных GET запросов с параметрами
- **Информация о стране** - получение данных о стране через REST Countries API с интерактивной картой
- **Случайная собака** - получение случайных изображений собак через Dog CEO API

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-username/название-репозитория.git
cd название-репозитория
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
- Windows:
```bash
venv\Scripts\activate
```
- Linux/Mac:
```bash
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

Запустите программу:
```bash
python main.py
```

Выберите действие из меню:
1. GET запрос по URL - консольный режим
2. Информация о стране - веб-интерфейс с картой
3. Случайная собака - веб-интерфейс с изображениями

## Технологии

- Python 3.11+
- Flask - веб-фреймворк
- Requests - HTTP библиотека
- Colorama - цветной вывод в консоль
- Leaflet.js - интерактивные карты

## API

Проект использует следующие публичные API:
- [REST Countries API](https://restcountries.com/) - информация о странах
- [Dog CEO API](https://dog.ceo/dog-api/) - изображения собак
