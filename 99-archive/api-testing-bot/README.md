# 🤖 Telegram AI Assistant Bot

Интеллектуальный Telegram-бот, который отвечает на вопросы пользователей, используя AI-модель Vitaliy от OpenRouter. Бот поддерживает естественный диалог и может помочь в решении различных задач.

## ✨ Особенности

- 🧠 **AI-powered** — использует модель Vitaliy от OpenRouter для ответов
- 💬 **Естественный диалог** — бот понимает контекст и отвечает на вопросы
- ⚡ **Асинхронная архитектура** — быстрая обработка сообщений
- 🛡️ **Обработка ошибок** — корректная работа при сбоях API
- 📝 **Логирование** — подробные логи для отладки
- 🔧 **Простая конфигурация** — управление через переменные окружения

## 🚀 Быстрый старт

### Предварительные требования

- Python 3.8+
- Telegram аккаунт
- API ключ от [OpenRouter](https://openrouter.ai/keys)
- Telegram Bot Token (получить у [@BotFather](https://t.me/botfather))

### Установка

#### 1. Клонирование репозитория
```bash
git clone <repository_url>
cd ТГ-бот\ тестирования\ API
```

#### 2. Создание виртуального окружения
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

#### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

#### 4. Конфигурация
Создайте файл `.env` в корневой папке проекта:
```env
BOT_TOKEN=ваш_telegram_bot_token
OPENROUTER_API_KEY=sk-or-v1-ваш_openrouter_ключ
```

**Где найти ключи:**
- **BOT_TOKEN** — напишите [@BotFather](https://t.me/botfather) в Telegram и создайте нового бота
- **OPENROUTER_API_KEY** — зарегистрируйтесь на [OpenRouter.ai](https://openrouter.ai) и получите ключ API

#### 5. Запуск бота
```bash
python main.py
```

Если всё работает, в консоли должно появиться:
```
2026-01-10 15:30:45,123 [INFO] app.config: ✅ BOT_TOKEN загружен успешно
============================================================
🚀 Starting Telegram Bot...
============================================================
✅ Подключено к боту: @YourBotName
Bot ID: 123456789
============================================================
📡 Начинается polling...
============================================================
```

## 🐛 Решение проблем

### Ошибка: "BOT_TOKEN не установлен"
**Решение:** Проверьте, что в файле `.env` правильно указан `BOT_TOKEN`:
```env
BOT_TOKEN=123456789:ABCDEfghijklmnopqrstuvwxyz_ABC
```

### Ошибка: "OPENROUTER_API_KEY не найден"
**Решение:** Добавьте ключ OpenRouter в `.env`:
```env
OPENROUTER_API_KEY=sk-or-v1-xxxxxxxxxxxxx
```

### Бот не отвечает на сообщения
**Решение:**
1. Проверьте интернет соединение
2. Убедитесь, что OpenRouter API работает (проверьте статус на [openrouter.ai](https://openrouter.ai))
3. Проверьте логи в консоли для ошибок

### Timeout при запросе к API
**Решение:** OpenRouter API может быть медленным. Попробуйте:
1. Отправить сообщение позже
2. Проверить статус OpenRouter
3. Увеличить timeout в `services.py` (по умолчанию 30 сек)
