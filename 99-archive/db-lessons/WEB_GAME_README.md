# SQL Игра - Веб-версия

## 🌐 Описание

Веб-версия SQL игры для обучения SQL запросам через браузер. Интерактивный интерфейс с проверкой запросов в реальном времени.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

Или установите Flask вручную:
```bash
pip install Flask
```

### 2. Подготовка базы данных

Убедитесь, что создана база данных `products_orders.db`:

```bash
# Создание таблиц
python create_tables_python.py

# Заполнение данными
python fill_products_python.py
```

### 3. Запуск веб-сервера

```bash
python app.py
```

### 4. Откройте в браузере

Перейдите по адресу: **http://localhost:5000**

## 📁 Структура файлов

```
├── app.py                    # Flask приложение
├── templates/
│   ├── index.html           # Главная страница
│   └── game.html            # Страница игры
├── static/
│   └── style.css            # Стили
├── sql_game_schema.sql      # Схема БД игры
├── sql_game_tasks.sql       # Задания
└── requirements.txt         # Зависимости Python
```

## 🎮 Как играть

1. **Откройте главную страницу** - http://localhost:5000
2. **Введите ваше имя** для сохранения статистики
3. **Выберите уровень** сложности (1-5)
4. **Читайте задание** и вводите SQL запрос в редакторе
5. **Нажмите "Проверить"** - система проверит ваш запрос
6. **Используйте подсказки** если застряли
7. **Смотрите результаты** - таблица с данными отобразится автоматически

## 🔧 API Endpoints

### GET `/`
Главная страница

### GET `/game?level=1`
Страница игры с заданиями уровня

### GET `/api/tasks?level=1`
Получение списка заданий уровня

### POST `/api/check`
Проверка SQL запроса
```json
{
  "task_id": 1,
  "query": "SELECT * FROM products;"
}
```

### GET `/api/stats`
Получение статистики игрока

### POST `/api/set_player`
Установка имени игрока
```json
{
  "name": "Игрок"
}
```

## 🐛 Решение проблем

### Ошибка: "ModuleNotFoundError: No module named 'flask'"
```bash
pip install Flask
```

### Ошибка: "База данных не найдена"
Убедитесь, что создали `products_orders.db`:
```bash
python create_tables_python.py
python fill_products_python.py
```

### Порт 5000 занят
Измените порт в `app.py`:
```python
app.run(debug=True, host='127.0.0.1', port=5001)
```

### Страница не загружается
Проверьте, что сервер запущен и нет ошибок в консоли.
