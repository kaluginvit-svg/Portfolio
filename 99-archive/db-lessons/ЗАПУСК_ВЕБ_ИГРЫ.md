# 🚀 Быстрый запуск веб-версии SQL Игры

## ✅ Flask установлен!

Flask успешно установлен. Теперь можно запустить веб-приложение.

## 📝 Пошаговая инструкция

### 1. Убедитесь, что база данных создана

```bash
# Если еще не создана, выполните:
python create_tables_python.py
python fill_products_python.py
```

### 2. Запустите веб-сервер

```bash
python app.py
```

Вы увидите сообщение:
```
============================================================
SQL ИГРА - ВЕБ-ВЕРСИЯ
============================================================

Откройте в браузере: http://localhost:5000
Для остановки нажмите Ctrl+C
```

### 3. Откройте в браузере

Перейдите по адресу: **http://localhost:5000**

## 🎮 Использование

1. **Введите ваше имя** на главной странице
2. **Выберите уровень** сложности (1-5)
3. **Читайте задание** и вводите SQL запрос
4. **Нажмите "Проверить"** для проверки ответа
5. **Используйте подсказки** если нужно

## ⚠️ Важно

- **Не закрывайте терминал** пока игра запущена
- Для остановки нажмите **Ctrl+C** в терминале
- Если порт 5000 занят, измените порт в `app.py`

## 🔧 Альтернативные команды

Если `pip` не работает, используйте:
```bash
python -m pip install Flask
```

Если `python` не работает, попробуйте:
```bash
py -m pip install Flask
py app.py
```

## 📱 Доступ с других устройств

Если хотите открыть игру с другого устройства в той же сети:

1. Найдите ваш IP адрес:
   ```bash
   ipconfig  # Windows
   ```

2. Измените в `app.py`:
   ```python
   app.run(debug=True, host='0.0.0.0', port=5000)
   ```

3. Откройте в браузере: `http://ВАШ_IP:5000`
