### Скриншот 2: Результат JOIN запроса

**Что показать:**
- Результат выполнения JOIN запроса из раздела 4.4
- Должны быть видны: номер заказа, имя клиента, название товара, категория, количество, цена

**Как сделать:**
1. Выполните запрос из раздела 4.4 файла `queries.sql`:
```sql
SELECT 
    o.order_number,
    o.customer_name,
    o.order_date,
    o.status,
    p.name AS product_name,
    p.category,
    oi.quantity,
    oi.price AS item_price,
    (oi.quantity * oi.price) AS line_total
FROM orders o
INNER JOIN order_items oi ON o.id = oi.order_id
INNER JOIN products p ON oi.product_id = p.id
ORDER BY o.order_date DESC, o.order_number;
```
2. Сделайте скриншот результата запроса

### Скриншот 3: Агрегация (GROUP BY + HAVING)

**Что показать:**
- Результат запроса агрегации из раздела 4.2
- Должны быть видны: категория, количество товаров, средняя цена, мин/макс цена

**Как сделать:**
1. Выполните запрос из раздела 4.2 файла `queries.sql`:
```sql
SELECT 
    category,
    COUNT(*) AS product_count,
    round(AVG(price), 2) AS avg_price,
    round(MIN(price), 2) AS min_price,
    round(MAX(price), 2) AS max_price,
    SUM(stock_quantity) AS total_stock
FROM products
GROUP BY category
HAVING COUNT(*) > 5
ORDER BY product_count DESC;
```
2. Сделайте скриншот результата

### Скриншот 4: Пример LIKE (поиск по шаблону)

**Что показать:**
- Результат поиска товаров с использованием LIKE и UPPER()
- Должны быть видны товары, название которых содержит "Смарт" (независимо от регистра)

**Как сделать:**
1. Выполните запрос из раздела 4.3 файла `queries.sql`:
```sql
SELECT 
    sku,
    name,
    category,
    price
FROM products
WHERE name LIKE '%Смарт%' OR name LIKE '%смарт%' OR UPPER(name) LIKE '%СМАРТ%'
ORDER BY price DESC;
```
2. Сделайте скриншот результата

### Скриншот 5: Топ-N товаров (SELECT + WHERE + ORDER BY + LIMIT)

**Что показать:**
- Результат запроса топ-10 самых дорогих товаров из раздела 4.1
- Должны быть видны: SKU, название, категория, цена, остаток на складе

**Как сделать:**
1. Выполните запрос из раздела 4.1 файла `queries.sql`:
```sql
SELECT 
    sku,
    name,
    category,
    price,
    stock_quantity
FROM products
WHERE stock_quantity > 0
ORDER BY price DESC
LIMIT 10;
```
2. Сделайте скриншот результата

## 💡 Подсказки

### Если что-то не работает:

1. **Проверьте, что включены внешние ключи:**
   ```sql
   PRAGMA foreign_keys = ON;
   ```

2. **Проверьте количество товаров:**
   ```sql
   SELECT COUNT(*) FROM products;
   ```
   Должно быть ≥100

3. **Проверьте наличие заказов:**
   ```sql
   SELECT COUNT(*) FROM orders;
   ```

4. **Если запросы не выполняются, проверьте синтаксис SQL**

После выполнения каждого раздела queries.sql проверяйте результаты соответствующими SELECT запросами.
