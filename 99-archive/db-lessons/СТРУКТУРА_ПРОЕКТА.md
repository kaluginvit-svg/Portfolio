# Структура проекта - Базы данных

## 📁 Организация файлов по базам данных

### 🛒 products_orders.db (Текущее задание)

**Создание структуры:**
- `create_tables.sql` - создание таблиц products, orders, order_items (с проверками)
- `create_tables_python.py` - Python альтернатива для создания таблиц

**Заполнение данными:**
- `setup_products_orders.sql` - основной скрипт (создание таблиц + заполнение 120 товаров)
- `fill_products_python.py` - Python скрипт для заполнения товарами (если SQL не работает)

**Работа с данными:**
- `queries.sql` - все SQL запросы для задания (INSERT, UPDATE, DELETE, SELECT, JOIN)

**Документация:**
- `ИНСТРУКЦИЯ_СКРИНШОТЫ.md` - инструкция по выполнению задания

### 🍳 recipes.db (Many-to-Many - Рецепты)

**Создание и заполнение:**
- `recipes.py` - Python скрипт для создания БД рецептов
- `recipes.sql` - SQL версия (создание таблиц + данные)

**Запросы:**
- `recipes_queries.sql` - примеры запросов для работы с рецептами

**Связанные таблицы:**
- `create_related_tables.sql` - создание таблиц orders и order_items для изучения JOIN (используется с recipes.db)

## 📝 Рекомендации по использованию

### Для выполнения текущего задания (products_orders.db):

1. **Создание таблиц:**
   ```bash
   # Вариант 1: SQL
   sqlite3 products_orders.db < create_tables.sql
   
   # Вариант 2: Python
   python create_tables_python.py
   ```

2. **Заполнение данными:**
   ```bash
   # Вариант 1: SQL (если работает)
   sqlite3 products_orders.db < setup_products_orders.sql
   
   # Вариант 2: Python (если SQL не работает)
   python fill_products_python.py
   ```

3. **Выполнение запросов:**
   - Откройте `queries.sql` в DBeaver
   - Выполните все запросы по порядку

### Для изучения SQL:

- Используйте файлы `SQL_*_Объяснение.md` для изучения операторов
- Примеры из `insert_1000_products.sql` и `insert_products.sql` для практики
