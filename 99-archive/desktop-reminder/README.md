# Десктоп-напоминалка (Windows 10, Python, Tkinter)

Простое приложение-напоминалка для Windows 10 с хранением в SQLite, всплывающими системными уведомлениями (win10toast) или резервным popup-окном Tkinter. Интерфейс позволяет добавлять, просматривать, фильтровать, отмечать и удалять напоминания.

## Возможности
- Хранение в SQLite, автоматическое создание таблицы.
- Статусы: `Ожидает`, `Готово`, `Просрочено`, `Отменено`.
- Автоперевод в `Просрочено`, если время прошло более чем на минуту.
- Фоновая проверка напоминаний (каждую секунду).
- Уведомления через Win10 Toast (если разрешены в системе) или popup Tkinter.
- Быстрое создание напоминания (+1/5/15 минут), фильтр по статусу, двойной клик для просмотра деталей.

## Структура
- `database.py` — класс `ReminderDatabase`, CRUD, выборки, статусные обновления.
- `notifications.py` — класс `NotificationManager`, фоновый поллинг и показ уведомлений.
- `gui.py` — запуск Tkinter UI и взаимодействие с БД/уведомлениями.
- `requirements.txt` — зависимости (win10toast).

## Формат даты/времени
Используется формат `YYYY-MM-DD HH:MM`, например `2025-12-31 14:30`.

## Установка и запуск (Windows, venv)
1) Создать и активировать виртуальное окружение:
```powershell
python -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
```
2) Установить зависимости:
```powershell
pip install -r requirements.txt
```
3) Запустить приложение (основной вход — `app.py`):
```powershell
python app.py
```

## Использование
- Введите название, описание и время, нажмите «Добавить».
- Фильтр по статусу — выпадающий список над таблицей.
- Двойной клик по записи — просмотр деталей.
- Кнопки: «Отметить как Готово», «Отметить как Отменено», «Удалить», «Тест уведомления».
- Кнопки «+1/5/15 мин» быстро подставляют время.
- При закрытии запросит подтверждение, остановит фоновые проверки и закроет БД.

## Примечания по уведомлениям
- Системные уведомления Win10 могут быть заблокированы политиками или настройками «Фокусировка внимания». В этом случае автоматически используется popup-окно поверх всех окон.
- Popup закрывается автоматически через 30 секунд.

## Минимальные версии/зависимости
- Python 3.8+ (рекомендуется).
- Windows 10 (для системных уведомлений через win10toast).
