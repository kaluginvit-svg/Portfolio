Напиши на Python приложение-напоминалку для Windows 10. Должно хранить все напоминания в базе данных SQLite3. У каждого напоминания есть:

Заголовок

Описание

Дата и время срабатывания

Метка статуса: "Ожидает", "Готово", "Просрочено", "Отменено"

При наступлении времени напоминание должно появляться поверх всех окон (popup-окно) или в виде системного уведомления Windows 10 (через win10toast или аналог).
Должна быть возможность:

Добавить новое напоминание

Удалить напоминание

Отметить как "Готово" или "Отменено"

Автоматически переводить просроченные в статус "Просрочено"

Просмотреть все напоминания с фильтром по статусу

Хранилище — SQLite3. При старте приложение должно проверять и создавать таблицы, если их нет.
Интерфейс можно сделать в tkinter, но уведомления должны работать даже если окно программы свернуто.
Код должен быть структурирован: отдельный файл для БД, отдельный для GUI/логики уведомлений.
