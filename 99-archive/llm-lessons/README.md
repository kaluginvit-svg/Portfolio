# Работа с LLM — короткий отчёт

## Запуск тестов из терминала

```bash
# Активировать venv (если есть)
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate

# Запрос через ProxyAPI (интерактивный ввод)
python proxyapi_request.py

# Опционально: прямой OpenAI
python openai_request.py --prompt "Ваш вопрос" --temperature 0.7 --max_tokens 1000
```

При каждом прогоне в консоли выводятся: ответ модели, модель, использованные токены, temperature и max_tokens — эти данные нужно переносить в таблицу ниже.
