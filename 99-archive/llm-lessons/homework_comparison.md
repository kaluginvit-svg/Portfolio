# Сравнительная характеристика ответов: планирование и разбор расходов

Два ответа модели в формате JSON — «Планирование расходов на неделю» (ответ 1) и «Еженедельный разбор личных расходов» (ответ 2). Ниже — сравнение по структуре, содержанию и пригодности к использованию.

---

## Ответ 1: «Планирование расходов на неделю»

```json
{
  "title": "Планирование расходов на неделю",
  "steps": [
    "шаг 1: Определить сумму доступных средств после обязательных платежей",
    "шаг 2: Выделить 10000 рублей на сбережения",
    "шаг 3: Рассчитать остаток на повседневные расходы",
    "шаг 4: Разделить остаток на количество дней недели",
    "шаг 5: Составить список необходимых покупок и расходов",
    "шаг 6: Установить лимит на ежедневные траты",
    "шаг 7: Отслеживать расходы ежедневно",
    "шаг 8: В конце недели подвести итог и скорректировать план"
  ],
  "notes": [
    "Доступно на расходы: 25000 рублей",
    "Сбережения — приоритетная цель недели",
    "Контролируйте ежедневные траты для достижения цели"
  ]
}
```

---

## Ответ 2: «Еженедельный разбор личных расходов»

```json
{
  "title": "Еженедельный разбор личных расходов",
  "steps": [
    "шаг 1: Открой выписку по карте или приложение с расходами за неделю.",
    "шаг 2: Быстро выпиши все траты по категориям (еда, транспорт, покупки, развлечения и т.д.).",
    "шаг 3: Отметь траты, которые были необязательными или импульсивными.",
    "шаг 4: Подсчитай сумму по каждой категории и общую сумму расходов.",
    "шаг 5: Сравни с прошлой неделей или своим бюджетом, выдели, где было больше всего лишнего.",
    "шаг 6: Запиши 1–2 простых улучшения на следующую неделю (например, меньше кофе навынос, готовить дома).",
    "шаг 7: Сохрани заметки и цели, чтобы вернуться к ним через неделю."
  ],
  "notes": [
    "Не трать больше 3 минут на каждый шаг.",
    "Будь честен с собой — это только для тебя.",
    "Маленькие улучшения работают лучше, чем жёсткие запреты."
  ]
}
```

## Сравнительная таблица

| Критерий | Ответ 1 | Ответ 2 |
|----------|--------|--------|
| **Фокус** | Планирование на будущую неделю (распределение денег) | Разбор уже совершённых трат и выводы |
| **Количество шагов** | 8 | 7 |
| **Ориентир по времени** | Нет | Есть («не больше 3 минут на шаг», общий чек-лист на 15–20 мин) |
| **Конкретика** | Числа из теста (10000, 25000) — привязаны к одному кейсу | Без конкретных сумм, применимо к любому пользователю |
| **Тон notes** | Нейтральный, инструктивный | Более личный, поддерживающий («будь честен с собой», «только для тебя») |
| **Практичность** | Хорош для «с нуля» планирования бюджета | Хорош для еженедельного ретро-анализа и привычки |

---

## Сильные стороны

**Ответ 1**
- Чёткая последовательность: от «сколько есть» → сбережения → остаток → лимиты → контроль.
- Явно заложены приоритет сбережений и ежедневный контроль.
- Удобен, когда уже известны доход и обязательные платежи и нужно разложить неделю по дням.

**Ответ 2**
- Соответствует запросу «разбор расходов»: выписка → категории → лишние траты → итог → улучшения.
- Ограничение по времени (3 мин на шаг) помогает уложиться в 15–20 минут.
- Notes дают мотивацию и установку («маленькие улучшения лучше запретов»), не только сухую инструкцию.
- Конкретные примеры улучшений (кофе навынос, готовить дома) облегчают применение.

---

## Слабые стороны / риски

**Ответ 1**
- Жёстко привязан к числам из тестового запроса (10000, 25000) — при другом доходе/цели шаги те же, но notes выглядят чужими.
- Нет привязки к уже совершённым тратам и разбору «куда ушло».
- Notes ближе к констатации, чем к подсказкам по поведению.

**Ответ 2**
- Нет явного шага «спланировать лимиты на следующую неделю» — только «записать улучшения»; связку «разбор → план» пользователь может дописать сам.
- «Сравни с прошлой неделей» предполагает, что разбор делается не в первый раз.

---

## Вывод

- **Ответ 1** лучше подходит для **планирования недели вперёд** при известном доходе и цели по сбережениям.
- **Ответ 2** лучше подходит для **еженедельного разбора**: куда ушли деньги, где лишнее, 1–2 улучшения на следующую неделю, в пределах 15–20 минут.

Для сценария «чек-лист для еженедельного разбора личных расходов за 15–20 минут» ответ 2 точнее попадает в задачу по смыслу, времени и тону. Ответ 1 полезен как дополнение — план на следующую неделю после того, как разбор по ответу 2 уже сделан.
