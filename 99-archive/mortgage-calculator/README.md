# Ипотечный калькулятор

Flask-приложение для расчета ипотеки/рассрочки: аннуитетный платеж, переплата, досрочные платежи (сокращение срока или платежа) и экспорт графика в Excel. Проценты считаются от суммы после вычета первоначального взноса; нулевая ставка поддерживается.

## Быстрый старт (Windows PowerShell)

1) Установите Python 3.11+  
2) Создайте и активируйте виртуальное окружение:  
   ```powershell
   python -m venv .venv
   . .\.venv\Scripts\Activate.ps1
   ```
3) Установите зависимости:  
   ```powershell
   python -m pip install --upgrade pip
   python -m pip install -r requirements.txt
   ```
4) Запустите dev-сервер (по умолчанию 127.0.0.1:5000):  
   ```powershell
   python -m flask --app app run --debug
   ```

После запуска откройте браузер на `http://127.0.0.1:5000`.

## Переменные окружения

Можно передать переменные окружения непосредственно в консоли PowerShell:

- `FLASK_APP` — точка входа приложения (по умолчанию `app.py`).
- `FLASK_DEBUG` — `1` для режима разработки с авто-перезапуском.
- `FLASK_RUN_HOST` — адрес интерфейса (например, `0.0.0.0` для внешнего доступа).
- `FLASK_RUN_PORT` — порт dev-сервера.

## Возможности

- Рассрочка: кнопка «Рассрочка» скрывает поле ставки и считает под 0%.
- Досрочные платежи: добавляйте несколько платежей, выбирая режим «Сокращать срок» или «Снижать платеж».
- Экспорт: кнопка «Сохранить в Excel» выгружает график платежей.
- Поддержка нулевой ставки: формула корректно обрабатывает случай без процентов.

## Структура

- `app.py` — Flask-приложение, API `/calculate` и `/export`, расчёт досрочных платежей.
- `templates/index.html` — страница калькулятора с барочным оформлением.
- `static/script.js` — логика формы, рассрочки, досрочных платежей и выгрузки в Excel.
- `static/styles.css` — стили барокко.

## Проверка

- Быстрая smoke-проверка: запустить сервер и ввести тестовые значения в форме; убедиться, что расчет отображается без ошибок.
- Для локального API-запроса можно использовать:
  ```powershell
  Invoke-RestMethod -Method Post -Uri http://127.0.0.1:5000/calculate `
    -ContentType "application/json" `
    -Body (@{ amount=3500000; years=20; rate=10.5 } | ConvertTo-Json)
  ```
