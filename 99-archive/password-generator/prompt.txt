Напиши CLI-приложение на Python для хранения паролей с шифрованием, используя SQLite3.
Требования:

При первом запуске программа должна запросить пароль входа, сохранить его в базе данных в виде хеша SHA-256.

При последующих запусках программа требует этот пароль для входа.

Все хранимые данные — в формате:

Название/откуда (например, "Google")

Логин

Пароль (в зашифрованном виде)

Для шифрования паролей используй Fernet из библиотеки cryptography.

Ключ для шифрования хранить локально в файле .key (если файла нет — генерировать при первом запуске).

Команды CLI:

add — добавить новую запись

get — получить пароль по названию (после ввода мастер-пароля)

list — показать список всех сохранённых записей (названия и логины)

delete — удалить запись по названию

new - создать новый пароль

Все данные хранятся в SQLite3, при старте проверять наличие таблиц и при необходимости создавать.

Код должен быть структурирован — функции для БД, функции для шифрования, основной CLI-обработчик.
