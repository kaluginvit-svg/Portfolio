# Telegram Bot для сбора анкет

Телеграм-бот на PyTelegramBotAPI для сбора анкет пользователей с сохранением в PostgreSQL базу данных.

## Структура проекта

- `bot.py` - основной файл с ботом и обработчиками сообщений
- `database.py` - модуль для работы с базой данных PostgreSQL
- `config.py` - модуль для управления конфигурацией
- `requirements.txt` - зависимости проекта
- `.env` - файл с переменными окружения (создается на основе .env.example)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example` и заполните его данными:
   - `TELEGRAM_BOT_TOKEN` - токен вашего Telegram бота (получить у @BotFather)
   - `DB_HOST` - хост базы данных PostgreSQL
   - `DB_PORT` - порт базы данных (по умолчанию 5432)
   - `DB_NAME` - имя базы данных
   - `DB_USER` - имя пользователя базы данных
   - `DB_PASSWORD` - пароль пользователя базы данных

## Запуск

```bash
python bot.py
```

## Функциональность

- `/start` - начать работу с ботом
- `/help` - показать список доступных команд
- `/status` - проверить статус подключения к базе данных и список таблиц
- `/form` - заполнить анкету (заглушка, будет реализовано позже)

## Архитектура

Проект разделен на модули:

- **bot.py** - содержит только логику бота и обработчики сообщений
- **database.py** - содержит весь код для работы с базой данных, включая:
  - Подключение к БД
  - Получение структуры таблиц из information_schema
  - Выполнение SQL запросов
- **config.py** - управление конфигурацией через переменные окружения

Структура таблиц получается напрямую из базы данных через `information_schema`, без ручного описания.
