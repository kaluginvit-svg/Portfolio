# Telegram-бот: поиск рецептов (TheMealDB)

Основа на aiogram: главное меню — поиск рецептов (TheMealDB) и список избранного. Поиск работает, избранное пока заглушка. Есть `/start`, `/ping`, приветствие на «привет». Кнопка «Открыть рецепт» выводит ингредиенты и инструкции прямо в чате; если есть видеорецепт — выводит ссылку.

## Быстрый старт
1. (Рекомендуется) создать и активировать виртуальное окружение:
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте `.env` рядом с `main.py` и добавьте:
   ```
   BOT_TOKEN=ваш_токен
   ```
4. Запуск:
   ```bash
   python main.py
   ```

## Что умеет
- `/start` — приветствие и меню: «Поиск рецептов», «Мои рецепты».
- `/ping` — возвращает `pong 🏓`.
- «Привет» в тексте — бот здоровается и показывает меню.
- Кнопки:
  - `🔎 Поиск рецептов` — просит ввести запрос, ищет через TheMealDB и показывает до 5 рецептов с кнопкой «В избранное».
  - `⭐ Мои рецепты` — заглушка для вывода сохранённых рецептов.
- Коллбеки:
  - `fav:add:<id>` — заглушка для сохранения в избранное (добавить БД).
  - `fav:del:<id>` — удаляет рецепт из избранного.
  - `open:<id>` — загружает ингредиенты и инструкции и показывает в чате, выводит видеоссылку.
  - `rate:<id>:<n>` — сохраняет оценку 1–5.
