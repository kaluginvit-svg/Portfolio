# Напоминалка для регулярных платежей

Telegram-бот, который собирает данные о регулярных платежах и напоминает за 1 день, 1 час и 30 минут до срока.

## Возможности
- Вопросы по шагам: название платежа, период (неделя/месяц/год), число платежей в период, сумма, ближайшая дата/время.
- Кнопки «Да/Нет» для добавления нескольких платежей подряд.
- Данные сохраняются в `data/payments.json`; при перезапуске расписание восстанавливается.

## Установка и запуск (Windows, PowerShell)
```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Создайте рядом с `bot.py` файл `.env`:
```
BOT_TOKEN=ВАШ_ТОКЕН
```

Запуск:
```
python bot.py
```

## Использование
- Напишите боту `/start`; сначала бот спросит, начинать ли ввод платежей, дальше — шаги опроса.
- Формат даты и времени: `ДД.ММ.ГГГГ чч:мм` (пример: `15.01.2026 18:30`).
- Напоминания приходят за 1 день, за 1 час и за 30 минут до платежа; затем дата автоматически переносится на следующий период.
