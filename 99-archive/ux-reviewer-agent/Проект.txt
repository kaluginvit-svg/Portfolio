🎯 Цель

Создать агента, который анализирует текст сайта и генерирует UX-отчёт с 5 рекомендациями по улучшению.

⚙️ Логика работы

1. Загружает HTML по ссылке.  
2. Извлекает текст и формирует промпт UX-эксперта.  
3. Отправляет текст в модель с системной ролью.  
4. Получает отчёт: достоинства, слабые места и 5 предложений по улучшению UX.

✅ Функциональные требования

- Агент вызывается через функцию `run(url)`.  
- Используется `gpt-4o` или `gpt-3.5-turbo`.  
- Возвращает структуру: плюсы, минусы, рекомендации.  
- Обработка ошибок API + повторы до 3 раз.

 🛡 Нефункциональные требования

- Python ≥ 3.10.  
- Чистый, типизированный код.  
- Зависимости: `openai`, `requests`, `beautifulsoup4`, `dotenv`, `tenacity`.

 📁 Структура проекта

ux_reviewer/
├── agent.py
├── openai_module.py
├── requirements.txt
└── README.md

 🧪 Приёмка

- Команда `python agent.py ` возвращает отчёт с рекомендациями.  
- Ответ обоснован и релевантен содержимому сайта.  
- Поведение при сбоях корректное.
